# CPK Genovese

超かぐや姫で出てきたジェノベーゼパスタの再現と分量計算プログラム
超かぐや姫にインスパイアされた非公式の個人レシピ

# Requirement

一般的なキッチン
フライパン
機能する水道

* spaghetti
* Genovese sauce

# Usage

詳細はrecipeにて記載
炒めて茹でて混ぜて削ったら完成

# Note

Genoveseの難易度は想像以上に難しかったので、人に提供する前に自分で試食をお勧めします
